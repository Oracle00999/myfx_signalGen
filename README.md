# Market Analyser Backend

This project is a backend service for analyzing forex market data and producing trading signals. It exposes an API built with Express, fetches market candles and live prices, calculates technical indicators, evaluates market structure, and turns that analysis into buy or sell signal ideas based on strategy rules.

The system also handles ongoing market operations around those signals. It can scan instruments across timeframes, store and monitor generated signals, run scheduled background jobs, and send Telegram alerts when notable trading opportunities or signal updates appear. In short, this codebase is the engine behind an automated forex market analysis and signal monitoring platform.
